# Firebase セットアップ手順

このアプリを動かすために必要な、Firebaseプロジェクトの作成と設定方法です。

## 1. プロジェクトの作成
1. [Firebase Console](https://console.firebase.google.com/) にアクセスします。
2. **「プロジェクトを追加」** をクリックします。
3. プロジェクト名を入力（例: `kadahira-gantt`）し、続行します。
4. Google アナリティクスは「無効」でも構いません。
5. **「プロジェクトを作成」** をクリックし、完了するまで待ちます。

## 2. Webアプリの登録とキーの取得
1. プロジェクトのトップページで、**`</>` (Web)** アイコンをクリックします。
2. アプリのニックネームを入力（例: `Kadahira Gantt App`）し、**「アプリを登録」** をクリックします。
3. **「Firebase SDK の追加」** という画面に、以下のようなコードが表示されます。
   ```javascript
   const firebaseConfig = {
     apiKey: "AIzaSy...",
     authDomain: "...",
     projectId: "...",
     storageBucket: "...",
     messagingSenderId: "...",
     appId: "..."
   };
   ```
4. この `firebaseConfig` の中身（`apiKey` から `appId` まで）をコピーしておきます。
5. エディタに戻り、`src/lib/firebase.js` を開き、ダミーの値をこのコピーした値に書き換えます。

## 3. Firestore データベースの作成
ここを飛ばすと、アプリからデータを保存しようとしたときにエラーになります。

1. Firebase Console の左メニューから **「ビルド」** > **「Firestore Database」** を選択します。
2. **「データベースを作成」** をクリックします。
3. **ロケーション**（場所）を選択します。日本なら `asia-northeast1 (Tokyo)` がおすすめです。
4. **セキュリティルール** の設定画面になります。
   - 開発中は **「テストモードで開始する」** を選択してください。
   - ※テストモードは30日間、誰でも読み書き可能になります。本番公開時にはルールを変更する必要があります。
5. **「作成」** をクリックします。

## 4. 完了
これで設定は完了です！
`npm run dev` でアプリを起動し、テキストボックスに入力して「登録」を押すと、エラーが出ずにリストに追加されるはずです。
Firestoreのコンソール画面を見ると、`tasks` というコレクションができ、データが増えているのが確認できます。
